WISEBOTS : Website for Chatbots

This was a project where we can give ideas about varied chatbots for industrial sectors.

Wisebot was meant for providing web interface & information about various chatbots which might be beneficial for websites which might need to integrate chatbots with websites.

It includes with rates of our chatbot and automatically sent to the client along with an interface for customers to be able to login and view.

Customers had the ability to view all their data with a dynamic search form, trend data, add notes to reports and even map their samples' physical location.

I worked on this system for 2 years before selling this to FOI Laboratories.
